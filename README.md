## Installazione

1. `git clone https://github.com/Mattia23Dev/Buka.git`
2. `cd Buka`
3. `npm install && npm run build && npm start`


## Uninstalling  

- Run `sudo dpkg -r Buka` or `sudo apt remove --purge buka`
